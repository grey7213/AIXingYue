# 制卡器交接功能需求（2026-07-16）

## 目标

- 对接 `card-maker-source-export-20260715.zip` 的互动角色卡能力：素材上传、世界书素材绑定、BGM、弹窗、悬浮窗、侧栏与场景切换。
- 新增独立 `Gemini` 站点模型分组，与现有 CelestiAI 模型并存；使用用户提供的服务端密钥，公开端不返回密钥或 Base URL。
- 所有模型选择位置显示当前固定价格 `50 惑梦币/次`，实际扣费仍以服务端 `CHAT_MESSAGE_COST` 为唯一权威。
- 创作中心不展示、导出或回传平台反扒世界书正文；反扒规则只在服务端运行时强制注入。
- 新增“本卡预设”：创作者可导入 SillyTavern Prompt Preset JSON，并显式决定该卡是否启用。
- 互动界面与本卡预设统一归类为“高级创作”：默认需惑梦农场 8 块土地全部解锁；管理员可对指定用户直接开放或收回白名单权限。

## 范围

- Web 创作中心、聊天页、共享 API、后端角色卡 CRUD/导入导出/聊天运行时。
- 本地媒体存储适配器使用服务器 `media-cache/card-assets`，执行上传意图、PUT、完成校验三阶段流程。
- 新 Gemini 分组使用提供方 `/models` 实测返回的 6 个模型；现有默认模型和历史角色引用不迁移。

## 验收标准

- ZIP Manifest、Node/Python 测试通过；新增代码 Python/JavaScript 语法检查通过。
- 新旧模型分组同时可见，选择 ID 唯一，均显示 `50 惑梦币/次`；新组至少一个模型真实回复成功且扣费 50。
- API、管理员读取、日志、Git 和前端均不出现模型 API Key。
- 公开详情、本人编辑详情、管理员详情、JSON/PNG 导出均不包含反扒条目 ID 或正文；聊天所有生成入口仍注入服务端真实反扒规则。
- 本卡预设关闭时保持旧卡行为；开启时按顺序注入 system-before/post-history，并且不受单会话“关闭全局预设”影响。
- 素材所有权、MIME、大小、SHA-256、文件魔数校验有效；跨用户完成/删除/绑定失败。
- 未满足农场全解锁且无管理员白名单的用户不能上传素材或保存高级创作字段；前端锁定提示与服务端 403 一致。
- 管理后台用户列表可查看高级创作来源（农场/管理员/未开放）并切换管理员白名单。
- 390px 与桌面浏览器完成创作页、模型选择器和互动运行时验收，无横向溢出、console/page error 为 0。
- 部署后 backend/Nginx active，内外 `/health` 为 OK，`CONTENT_MODE=local_only`，SQLite `quick_check=ok`。

- 不允许作者 JavaScript 接触主站 DOM、token、localStorage 或同源 API。
