# AI星月 SillyTavern 能力补齐需求

Updated: 2026-06-26

## 目标

在现有 AI星月 Web/App Shell 基础上，按优先级补齐 SillyTavern 高级玩家常用能力，同时保持普通用户可以低门槛创建角色卡并聊天。

## 范围

- 修复每日签到无限领取积分漏洞。
- 补齐聊天流式输出、Prompt Manager、世界书增强、角色卡兼容、群聊、用户自带 Key、扩展/记忆/媒体等能力。
- 管理员后台继续集中管理站点模型预设，用户前台不暴露站长 API Key。
- `CONTENT_MODE=local_only` 保持不变，除非后续明确要求变更内容源。

## 验收标准

- 每日签到在服务端按北京时间自然日幂等，一天最多加一次积分，旧接口和新奖励接口一致。
- 聊天页优先使用流式端点，前端能接收 SSE `start` / `delta` / `message_end` 并增量显示。
- 每个后续 SillyTavern 能力都需要有对应 API/UI 验证记录。

## 非目标

- 不在本轮重打 APK。
- 不在用户前台泄露管理员模型 API Key。
- 不为了补功能破坏现有角色库、注册登录、充值兑换码和管理员后台。
