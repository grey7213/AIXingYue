# RP Hub 互动卡与官推卡池需求（2026-07-16）

## 目标

- 兼容用户提供的 RP Hub/Tavo 风格 PNG 角色卡：Regex 生成的 Vue 界面、卡内弹窗和远程 BGM 在受限沙箱内可用。
- 保持现有安全边界：卡片不能读取主站 DOM、账号 token、主页面 localStorage，也不能任意联网或加载任意第三方脚本。
- 管理后台按角色维护“官推卡池”；首页“官方每周推荐”每次刷新从池内随机展示，且不与下方普通推荐重复。
- 继续使用现有高级创作授权：管理员可按用户开放/收回，未授权用户不能上传平台托管素材或保存高级创作字段。

## 验收标准

- 目标 PNG 的 `ui:status` Regex 界面可显示；弹窗按钮可打开/关闭；悬浮音乐播放器可见。
- 仅允许固定 Vue CDN、Font Awesome CSS 和 `raw.githubusercontent.com` 音频；其他外部脚本、样式、图片、媒体仍被移除或 CSP 阻止。
- 音频播放遵守浏览器用户手势策略，不自动绕过 autoplay 限制。
- sandbox 不包含 `allow-same-origin`，卡片不能读取父页面或账号数据，既有 Tavo 安全回归继续通过。
- `local_apps` 持久化官推标记；只有管理员可修改。
- 后台角色列表可筛选官推池、加入/移出池，并显示官推状态。
- 首页首次加载和“换一批”使用新的 seed，从公开且已发布的官推池随机取最多 2 张；池为空时回退普通列表前两张。
- 官推卡从普通推荐区排除；私有、草稿或已删除卡不会出现在官推位。
- 后端/前端语法检查、临时 SQLite 验证、桌面与 390px 浏览器验证通过；部署后服务、Nginx、健康检查、`CONTENT_MODE=local_only` 和 SQLite `quick_check` 正常。

## 非目标

- 不允许任意 CDN、任意音频域名或卡片脚本访问主站 API。
- 不重建 APK。
- 不重复实现已有高级创作权限模型。
