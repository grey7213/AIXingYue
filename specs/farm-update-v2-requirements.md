# 惑梦农场 V2 更新需求

## 目标

- 审计用户新提供的 `E:\obs录制\农场.zip`，确认它是否为现有惑梦农场的新版本。
- 若压缩包确为新版农场，则在不影响用户对话、充值、签到历史和余额的前提下更新现有农场页面及必要接口。

## 验收标准

- 先以压缩包真实源码、构建产物和 README 为准识别功能，不凭文件名推断。
- 保持服务端农场状态权威、账号鉴权、幂等键和每日奖励唯一约束。
- 不恢复 localStorage 权威状态、用户 API Key 输入或浏览器自行计算奖励。
- 桌面和 390px 移动端无溢出、无控制台错误。
- 充值、订单、爱发电、在线支付宝、兑换码、聊天记录和旧 APK 签到接口不受影响。
- 不再展示或生成系统 NPC/虚构好友；真实好友关系未接入时显示明确空状态。
- 作物经济继续使用服务端权威配置 `50→90 / 100→190 / 180→340`，不采用原型硬编码收益。

## 非目标

- 不因新版视觉包重置或迁移现有用户农场数据，除非压缩包明确要求且完成兼容迁移设计。
- 不重打 APK。
- 不在本次新增真实好友关系表或玩家间资产转移。
