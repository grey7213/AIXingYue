# AI星月 Web 首帧防闪与主题统一 Requirements

Updated: 2026-07-11

## 目标

- 修复所有 Web App 页面在刷新、点击导航后短暂显示 Alpine 指令、字面 `\n`、未初始化数字或模板内容的问题。
- 修复创作工坊标题的双重换行转义。
- 将创作工坊、信息中心统一为 AI星月暖纸橙金 / 暖黑主题，消除紫色旧皮肤残留和低对比。

## 验收

- 所有带 `x-cloak` 的页面在 Alpine 接管前不可见；共享 CSS 内有本地 `[x-cloak]` 规则。
- HTML Alpine 表达式不再包含双转义 `\\n`；运营文案中的历史字面 `\n` 能安全还原为换行。
- Hub 页面不再重复执行 `init()`；工坊/信息中心加载完成前显示骨架，不闪 `0`。
- 信息中心显示公开官方角色数量，不再使用已过时的 upstream 数量。
- 375、768、1440 宽度无横向溢出，console error 为 0。

## 非目标

- 不改 APK，不改角色卡正文、标签或聊天逻辑。
