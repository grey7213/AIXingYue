# AI星月后台运营配置需求

## 目标

- 管理员可以可视化编辑用户前端能看到的关键运营内容，减少改代码和重部署的频率。
- 优先覆盖最常改的内容：官网首屏文案、下载区文案、App 全局公告、奖励签到额度、兑换/购买说明、推荐套餐。
- 保持当前 AI星月深色玻璃风格和已有导航，不重做前台视觉。

## 范围

- 官网首页：
  - 顶部副标语、运行状态文案。
  - 首屏标题、副标题、第二行说明。
  - 主按钮/副按钮文案和链接。
  - 信任栏文案、预览区标题/说明、下载区标题/说明、页脚服务文案。
  - 核心功能区标题、说明、功能卡片标题和描述。
  - 下载区版本/系统要求/架构/语言等信息卡与安装提示。
  - 常见问题标题、问题和回答。
- App Shell：
  - 管理员可开启/关闭全局公告。
  - 公告支持标题、正文、链接文案和链接地址。
  - 公告显示在 `/app` 各页面主内容顶部。
- App 信息中心：
  - 管理员可编辑信息中心标题、下载按钮文案、主标题、说明和统计项标签。
- 登录/用户中心/App 常用文案：
  - 管理员可编辑登录/注册页的副标语、Tab、按钮和提示。
  - 管理员可编辑用户中心和“我的”页的余额、签到、账户、人设、模型连接器和 API 信息说明。
  - 管理员可编辑 Web App 桌面侧栏和移动底栏导航标签。
  - 管理员可编辑探索、收藏、历史、我的角色、操作记录、兑换记录、工坊和图片聊天等主要空状态/占位文案。
  - 管理员可编辑 App 首页/探索页的搜索框、筛选标签、排行榜、排序、无图模式、卡片兜底、加载按钮。
  - 管理员可编辑用户中心登录/注册字段标签、输入占位、登录/注册/toast、签到/兑换/toast。
  - 管理员可编辑“我的”页人设字段、模型连接器字段/按钮、签到/兑换/退出/toast。
  - 管理员可编辑“我的角色”页面标题、卡片按钮、编辑弹窗、保存/删除/toast。
  - 管理员可编辑 App Shell 用户卡兜底昵称、积分后缀和进入“我的”的 title。
- 角色/聊天/创作/群聊：
  - 管理员可编辑角色详情页的标题、按钮、徽标、空态和分区标题。
  - 管理员可编辑聊天页的会话列表、记忆抽屉、消息操作、输入框、空态、确认/报错提示等主要可见文案。
  - 管理员可编辑角色编辑器的工具栏、基础字段、提示词/世界书/高级提示词/模型/占位分区的标题、提示、下拉选项和占位符。
  - 管理员可编辑群聊页的列表、创建面板、输入框、按钮、空态、toast 和确认提示。
- 奖励和购买：
  - 每日签到积分可配置。
  - 推荐任务文案和积分可配置。
  - 奖励页 Credits 标题、余额说明、套餐区标题、任务状态、兑换记录标题/说明/刷新、领取按钮文案可配置。
  - 爱发电/购买链接可配置。
  - 汇率说明、购买说明、推荐套餐可配置。
- 后台：
  - 新增“运营配置”页，使用表单编辑上述内容。
  - 支持保存后前台立即读取生效。

## 验收标准

- 后端提供公开读取 API：`GET /console/api/public/site-settings`。
- 后端提供管理员读写 API：`GET/POST /admin/api/site-settings`。
- `deposit_meta_json()`、`/console/api/web/rewards`、每日签到接口使用后台配置。
- 官网首页加载公开配置并替换关键文案/链接。
- 官网首页功能卡片、下载信息和 FAQ 可由后台配置替换。
- App Shell 加载公开配置并渲染公告。
- App 信息中心文案和统计标签可由后台配置替换。
- 登录/注册、用户中心、“我的”、App 导航标签、主要空状态和图片聊天占位文案可由后台配置替换。
- 角色详情、聊天页、角色编辑器和群聊页的主要用户可见文案可由后台配置替换。
- App 首页/探索筛选、“我的角色”管理弹窗、用户中心/“我的”页表单字段和交互 toast 可由后台配置替换。
- 后台“运营配置”页能保存并重新加载配置。
- 补漏审计后的剩余普通用户可见文案（图片回复标题、聊天空态连接词、Prompt Manager 注入位置选项、角色读取失败提示）可由后台配置替换。
- 本地和线上验证：保存测试公告/每日奖励/购买链接，前台 API 和页面能看到新值；测试后恢复默认或安全值。

## 非目标

- 本轮不做任意 HTML 富文本编辑，避免 XSS 和排版失控。
- 本轮不做图片资源上传/裁剪，图片仍通过现有静态资源和角色封面流程管理。
- 本轮不做服务器重启、Nginx、证书、域名等高危运维面板。
