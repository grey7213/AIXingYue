# AI星月 Web 全站 UI 改造需求

## 目标

以 `E:\obs录制\创作（角色卡编辑器）.zip` 内原型为主要视觉参考，统一网页端首页、探索、角色详情、聊天、创作工具、我的、历史、收藏、信息中心和管理后台的配色、比例、间距与组件质感。

## 验收标准

- 保留现有功能、数据接口和 AI星月品牌，不改 APK。
- 桌面端和移动端均无横向滚动、遮挡、文字溢出。
- 表单、卡片、按钮、标签、抽屉、顶部栏使用统一设计令牌。
- 375、768、1024、1440px 四个断点进行真实页面截图验证。
- 浅色主题文字对比度、键盘焦点和 reduced-motion 可用。

## 非目标

- 不照搬参考原型的业务字段或调试模拟功能。
- 不重写后端，不改变角色卡数据格式。
